Home Kubernetes Cluster for Real Estate Application

I have set up a Kubernetes cluster at home to practice and host a real estate application.

Deployment Workflow

GitHub Actions: Automates the build process and creates a Docker image.

ArgoCD: Deploys the application to my Kubernetes cluster.

Monitoring

Prometheus: Collects real-time metrics from the application.

Grafana: Visualizes metrics through dashboards.

Infrastructure

Running on ARM64 architecture with persistent storage and a load balancer for traffic management.
